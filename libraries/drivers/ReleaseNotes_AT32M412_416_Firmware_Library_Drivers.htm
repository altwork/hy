Release Notes for

AT32M412_416 Firmware Library Drivers

V2.0.2-2025/11/06

1.         CMP

l  更新cmp_filter_config()函数，设置FALSE时强制滤波值为0。

2.         CAN

l  修正部分寄存器位域定义，删除多余定义。

3.         TMR

l  更新tmr_interrupt_flag_get()函数，增加TMR_BRK2_FLAG的参数配置。

l  更新tmr_pwm_input_config()、tmr_encoder_mode_config()和tmr_output_channel_config()函数。通道模式配置前关闭通道使能，否则通道模式无法切换。

4.         CRM

l  新增crm_interrupt_flag_get()函数。

5.         更新部分注释。
